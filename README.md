A simple repairkit script for fivem QBcore

add item in item.txt into qb-core/shared/items.lua
['repairitem'] 					 = {['name'] = 'repairitem', 			 	  	['label'] = 'repairitem', 				['weight'] = 2500, 		['type'] = 'item', 	['image'] = 'repairkit.png', 			['unique'] = false, 	['useable'] = true, 	['shouldClose'] = true,	   ['combinable'] = nil,   ['description'] = 'self repair item for cars and trucks'},

Has a 5-10 second timer and removes item after repair is complete
